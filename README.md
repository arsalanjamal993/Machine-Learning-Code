<h1>Machine Learning with Scikit-Learn</h1>

<p>This repository showcases a complete <strong>Machine Learning journey</strong> using <strong>Scikit-Learn</strong>. It combines theory, practical Jupyter notebooks, datasets, and real-world projects to help you learn the ML pipeline from scratch.</p>

<h2>Overview</h2>
<p>The repository is structured to guide you from data preprocessing to building supervised models and evaluating them with standard metrics. It also includes an end-to-end project for predicting student scores, giving hands-on experience with ML workflows.</p>

<h2>Folder Structure</h2>
<pre>
Scikit_learn/
│
├── Data_Preprocessing/
│   ├── data_preprocessing.ipynb
│   ├── data_preprocessing_notes.txt
│   └── Sample_Data.csv
│
└── Supervised_ML/
    ├── Linear_Regression/
    │   ├── linear.ipynb
    │   └── Linear_regression_Notes.txt
    ├── Logistic_Regression/
    │   ├── logistic.ipynb
    │   └── logistic_regression_notes.txt
    ├── Decision_Tree/
    │   └── decision.ipynb
    └── KNN_Classification/
        ├── KNN.ipynb
        └── K_Nearest_Neighbors_KNN.txt
</pre>

<h2>Topics Covered</h2>
<ul>
    <li>Data Preprocessing (missing data, encoding, scaling, splitting)</li>
    <li>Linear Regression</li>
    <li>Logistic Regression</li>
    <li>Decision Trees</li>
    <li>K-Nearest Neighbors (KNN)</li>
    <li>Model evaluation metrics (Accuracy, Precision, Recall, F1-score, MAE, MSE, RMSE)</li>
    <li>End-to-end project: Student Score Prediction</li>
</ul>

<h2>Getting Started</h2>
<p>Clone the repository and run the notebooks locally:</p>
<pre>
git clone https://github.com/&lt;your-username&gt;/Machine-Learning-With-Scikit-Learn.git
cd Machine-Learning-With-Scikit-Learn/Scikit_learn
jupyter notebook
</pre>

<p>Each module includes:</p>
<ul>
